- Wayland Compositor - Hyprland 
- Shell - Zsh with oh-my-zsh
- Terminal - Kitty
- Bar - Waybar or Astal,AGS
- Spotify rice - Spicetify, pywal-spicetify
- Themes - Pywal and Oomox
- Wallpaper from wallhaven
- IDE - Vscodium with pywal extension
